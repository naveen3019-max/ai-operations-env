READY TO USE
===================================================================================

QUICK START:
   1. Install dependencies:
      pip install -r requirements.txt

   2. Run baseline evaluation:
      python baseline/run.py

   3. Try examples:
      python examples.py

   4. Build Docker image:
      docker build -t ai-operations-env .
      docker run ai-operations-env

IMPORT ENVIRONMENT:
   from env.environment import AIOperationsEnvironment
   env = AIOperationsEnvironment(seed=42, max_steps=100)
   obs = env.reset()

DEFINE CUSTOM AGENT:
   Follow BaselineAgent pattern in baseline/agent.py
   Implement select_action(observation) method
   Use task.reset(), task.env for running episodes

EVALUATE WITH TASKS:
   from tasks.easy import EasyEmailClassificationTask
   task = EasyEmailClassificationTask(max_steps=50)
   env = task.setup_environment()
   # Run agent
   result = task.evaluate(env)  # Returns TaskResult with score
